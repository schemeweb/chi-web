# chi-web

Portable (R7RS) versions of the comprehensive Chicken Scheme web eggs,
most of which are written by Peter Bex.

## SRFI dependencies

* SRFI 1
* SRFI 13
* SRFI 14
* SRFI 18
* SRFI 69

## Language dependencies

* defstruct
* matchable

## Library dependencies

* base64
* simple-md5
* simple-sha1
* sendfile

## The web stack

* uri-generic
* uri-common
* intarweb
* http-client
* spiffy
