# chi-web

Portable (R7RS) versions of the comprehensive Chicken Scheme web eggs,
most of which are written by Peter Bex.

* SRFI 1
* SRFI 13
* SRFI 14
* SRFI 18
* SRFI 69

* defstruct
* matchable

* base64
* simple-md5
* simple-sha1
* sendfile

* uri-generic
* uri-common
* intarweb
* http-client
* spiffy
